
2. **Move Unreleased items** to the new version
   - Cut from Unreleased ‚Üí Paste into new version

   ```markdown

   [1.1.0]: https://github.com/.../compare/v1.0.0...v1.1.0
   ```

4. **Tag the release** on GitHub
   ```bash
   git tag -a v1.1.0 -m "Release v1.1.0"
   ```
